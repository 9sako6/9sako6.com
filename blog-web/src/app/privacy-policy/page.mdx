# プライバシーポリシー

本サイトでは、以下のツールを導入しています。

- Cloudflare Web Analytics
  - 送信される利用者情報：ファーストパーティの Cookie、デバイス / ブラウザに関連するデータ、IP アドレス、サイト内 / アプリでのアクティビティ
  - サービス提供者：Cloudflare
  - 利用目的：アクセス解析
- Google Analytics
  - 送信される利用者情報：ファーストパーティの Cookie、デバイス / ブラウザに関連するデータ、IP アドレス、サイト内 / アプリでのアクティビティ
  - サービス提供者：Google
  - 利用目的：アクセス解析
